# .dotfiles

## Introduction

These are my dotfiles. They include configurartion files and some small scripts for setting up my machines. In addition to that, I added some minimalist wallpaper in the wallpaper-folder and some avatars in the avatar-folders.

## Description of folders

### aerospace

Config-files for aerospace, the macos-tilling manager. I should move that sooner or later into the mac-config specific folder.

### alacritty

Settings and themes for Alacritty, the best terminal-program in the world

### asbplayer

Configuration for the asb-player extension. As of the point of writing, this extension only seems to work in Chrome.

### avatars

Some avatars.

### bookmarks

Regular backup of my browser-bookmarks

### fedora-scrips

Configuration-script for my fedora installation. Work in Progress.

### git

My basic git configuration files.

### macos_config

Files specific to macOS. Includes my Brewfile and specific ssh-file.

### macos_scripts

Configuration and automation-scripts for a fresh macOS installation

### mpvacious

Configuration for mpvacious, the best supporting tool for sentence mining with mpv

### tmux

My tmux configuration. Work in prograss as of now.

### vim

My current vim configuration. If I ever bother learning it.

### wallpaper

Collection of my favorite wallpapers.

### windows_scripts

Scripts for configuring a basic windows installation. As of now, no longer maintained.

### yomitan

Backup of my yomitan config.

### yt-dlp

My yt-dlp config file for downloading content mostly for language learning.

### zsh

My .zshrc.
